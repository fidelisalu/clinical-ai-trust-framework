# Contributing

Thank you for considering contributing.

## What’s welcome
- Improvements to metric definitions
- Additional audit templates
- Example cases and annotated datasets
- Diagram improvements
- Clarity edits

## What’s not welcome
- Medical advice or diagnosis claims
- Any patient-identifiable data
- Claims that imply certification or regulatory approval

## How to contribute
1) Fork the repo
2) Create a branch
3) Submit a PR with a clear description of changes and motivation
